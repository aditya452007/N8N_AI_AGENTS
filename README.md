# n8n AI Agents

This repository contains a collection of custom **AI-powered agents built on n8n**, designed to automate workflows, integrate with APIs, and enhance productivity. Each agent focuses on real-world use cases like content creation, data processing, and intelligent task automation.

---

## üöÄ Available Workflows

This repository is organized into two main categories:

- **[POWER AUTOMATE](./POWER%20AUTOMATE)**: Contains workflows and resources related to Microsoft Power Automate.
- **[WORKFLOWS](./WORKFLOWS)**: A collection of n8n workflows for various services and platforms.

### WORKFLOWS

Below is a list of the available workflows in the `WORKFLOWS` directory:

- **[AI-AGENT-KIT](./WORKFLOWS/AI-AGENT-KIT)**: A kit for building AI agents.
- **[Discord](./WORKFLOWS/Discord)**: Workflows for Discord automation.
- **[Free Courses List](./WORKFLOWS/Free%20Courses%20List)**: Workflows for fetching and managing lists of free courses.
- **[Github](./WORKFLOWS/Github)**: Workflows for GitHub automation.
- **[Google Calendar](./WORKFLOWS/Google%20Calendar)**: Workflows for Google Calendar automation.
- **[Image](./WORKFLOWS/Image)**: Workflows for image processing.
- **[Linkedin](./WORKFLOWS/Linkedin)**: Workflows for LinkedIn automation.
- **[Prompting Agent](./WORKFLOWS/Prompting%20Agent)**: An agent for generating prompts.
- **[Reddit](./WORKFLOWS/Reddit)**: Workflows for Reddit automation.
- **[TELEGRAM](./WORKFLOWS/TELEGRAM)**: Workflows for Telegram automation.
- **[Youtube](./WORKFLOWS/Youtube)**: Workflows for YouTube automation.

---

## üìñ Usage

To use these workflows, you will need to have a running instance of n8n. You can then import the workflow files (`.n8n`) into your n8n instance.

Please refer to the `README.md` file in each workflow's directory for specific setup and usage instructions.
